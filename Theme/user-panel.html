Setup Guide
v2rayNG
Recommended client for Android
  1. 1
    Install v2rayNG from Google Play or download the APK from GitHub releases.
  2. 2
    Open the app, tap the + icon, then select Import config from clipboard after copying your subscription link above.
  3. 3
    Tap the connect button at the bottom of the screen. You're all set! 🎉

Streisand
Recommended client for iOS
  1. 1
    Download Streisand from the App Store (requires a non-restricted Apple ID).
  2. 2
    Copy your Auto Connect subscription link, open Streisand, and it will detect the link automatically.
  3. 3
    Select a server from the list and tap Connect. Done! 🚀

Hiddify Next
Recommended client for Windows
  1. 1
    Download Hiddify Next from the official website or GitHub releases page.
  2. 2
    Open the app, go to Add Profile, paste your subscription link and click Add.
  3. 3
    Select a proxy from the list, then click the Connect button. Enjoy! ✨
Download Hiddify Next
Hiddify Next
Recommended client for macOS
  1. 1
    Download Hiddify Next for macOS from GitHub or use Homebrew: brew install hiddify.
  2. 2
    Launch the app, click New Profile, and paste your subscription link from above.
  3. 3
    Pick a server and toggle the connection on. You're connected! 🍎

Hiddify Next
Recommended client for Linux
  1. 1
    Download the .AppImage or .deb package from the Hiddify Next GitHub releases.
  2. 2
    Open the app, navigate to Profiles, click Add and paste your subscription link.
  3. 3
    Select a node and press Connect. Enjoy your connection! 🐧
